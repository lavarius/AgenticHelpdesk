# Agentic Helpdesk

AI-powered ticket management system for student support.

## Project Structure

- `client/` — React + TypeScript + Tailwind CSS frontend (Vite, port 5174)
- `server/` — Node.js + Express + TypeScript backend (port 3001)
- Monorepo managed with Bun workspaces

## Documentation

Use context7 MCP server to fetch up-to-date documentation for libraries and frameworks before implementing features. This applies to all dependencies including React, Express, Prisma, Tailwind CSS, and any others. Do not rely solely on training data — always verify with current docs.

## Key Commands

- `bun install` — install all workspace dependencies from root
- `bun run dev` — start dev server (run from client/ or server/)
