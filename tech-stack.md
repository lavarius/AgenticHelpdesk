# Tech Stack

## Frontend
- **React** with **TypeScript**
- **Tailwind CSS** for styling
- **React Router** for navigation

## Backend
- **Node.js** with **Express** (TypeScript)
- **Database sessions** for authentication

## Database
- **PostgreSQL**

## ORM
- **Prisma**

## AI
- **Anthropic Claude API** — ticket classification, summaries, and suggested replies

## Email
- **SendGrid** or **Mailgun** — outbound replies and inbound webhooks for receiving support emails

## Deployment
- **Docker** + cloud provider (AWS, Railway, Fly.io, etc.)
